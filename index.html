Métodos Numéricos

Aplicación para calcular raices y errores

Esta página consta de 4 programas. En los cuales uno funciona para calcular errores. El resto de estos progranas consiste en econtrar raices por 3 metodos diferentes: Bisección, aproximaciones sucesivas y el metodo de Newton Raphson.

Programa 1

Este programa calcula 3 tipos de errores (Relativo, absoluto y porcentual) de un dato real y uno obtenido.

Programa 2

Este programa calcula la raiz de una función ingresada por el usuario por el metodo de "Bisección"

Programa 3

Este programa calcula la raiz de una función ingresada por el usuario por el metodo de "Aproximaciones Sucesivas"

Programa 4

Este programa calcula la raiz de una función ingresada por el usuario por el metodo de "Newton Raphson"
